# Hockey ID CRM — Архитектура системы

CRM для хоккейной школы «Hockey ID». Родители используют отдельное приложение.

---

## Технологический стек

| Технология | Назначение |
|------------|------------|
| **Next.js 14** | App Router, SSR/SSG |
| **React** | UI-компоненты |
| **TailwindCSS** | Стили, тёмная тема, неоновые акценты |
| **Prisma** | ORM |
| **PostgreSQL** | База данных |

---

## Разделы CRM

### 1. Dashboard (Главная)

- Количество игроков
- Количество команд
- Ближайшие тренировки
- Неоплаченные счета
- Посещаемость команды
- Последние рекомендации тренеров

**Маршрут:** `/dashboard`

---

### 2. Школа

- Настройки школы
- Возрастные группы
- Управление командами

**Маршрут:** `/schools`, `/schools/[id]`

**Модели:** `School`, `Team`

---

### 3. Команды

- Список команд
- Возрастная группа
- Главный тренер
- Список игроков
- Расписание команды
- Статистика команды

**Маршрут:** `/teams`, `/teams/[id]`

**Модели:** `Team`, `Coach`, `Player`, `Training`

---

### 4. Игроки

Карточка игрока:

| Секция | Поля |
|--------|------|
| **Профиль** | ФИО, фото, дата рождения, позиция, хват, рост, вес, команда |
| **Паспорт** | номер паспорта, международный ID, дата выдачи, срок действия |
| **Навыки** | скорость, бросок, дриблинг, выносливость |
| **Медицина** | последняя проверка, травмы, ограничения |
| **Статистика** | игры, голы, передачи, очки, штрафы |
| **История команд** | команда, сезон, лига |
| **Достижения** | название, год |
| **Видео** | ссылки |
| **Оплаты** | месяц, сумма, статус |
| **Посещаемость** | статусы тренировок |
| **Оценки тренеров** | рейтинг, рекомендации |
| **Рекомендации** | текст от тренера |

**Маршрут:** `/players`, `/players/[id]`, `/players/[id]/edit`

**Модели:** `Player`, `Passport`, `Skills`, `Medical`, `PlayerStat`, `TeamHistory`, `Achievement`, `Video`, `PlayerPayment`, `Attendance`, `CoachRating`

---

### 5. Тренеры

- ФИО
- Специализация
- Команды
- Проведённые тренировки
- KPI тренера

**Маршрут:** `/coaches`, `/coaches/[id]`

**Модели:** `Coach`, `Team`, `Training`, `CoachRating`

---

### 6. Расписание

Календарь тренировок:

- Команда
- Тренер
- Дата и время
- Место
- Список игроков
- Посещаемость

**Маршрут:** `/schedule`, `/trainings`, `/trainings/[id]`

**Модели:** `Training`, `Attendance`, `Team`, `Coach`

---

### 7. Финансы

- Список платежей
- Задолженности
- Оплата по командам
- Оплата по школе

**Маршрут:** `/payments`

**Модели:** `PlayerPayment`, `Player`, `Team`

---

### 8. Аналитика

- Прогресс игроков
- Посещаемость
- KPI тренеров
- KPI команды
- Статистика школы

**Маршрут:** `/analytics`

---

### 9. Коммуникации

- Обращения родителей
- Ответы тренеров
- История сообщений

**Маршрут:** `/communications`

**Модели:** `Message`, `Parent`, `User`

---

### 10. Настройки

- Роли: администратор, главный тренер, тренер
- Профиль пользователя

**Маршрут:** `/settings`

**Модели:** `User`, `UserSettings`

---

## Prisma модели

| Модель | Назначение |
|--------|------------|
| **School** | Школа |
| **Team** | Команда (ageGroup, coachId) |
| **Player** | Игрок |
| **Coach** | Тренер |
| **Training** | Тренировка |
| **Attendance** | Посещаемость (trainingId, playerId, status) |
| **PlayerPayment** | Оплата (month, year, amount, status) |
| **Skills** | Навыки (speed, shotAccuracy, dribbling, stamina) |
| **Medical** | Медицина (lastCheckup, injuries, restrictions) |
| **Passport** | Паспорт (passportNumber, internationalID, issueDate, expiryDate) |
| **Achievement** | Достижения |
| **Video** | Видео игрока |
| **CoachRating** | Оценки тренеров |
| **Message** | Сообщения родитель ↔ тренер |
| **User** | Пользователь (роль: SCHOOL_ADMIN, COACH, PARENT) |
| **Parent** | Родитель |
| **Notification** | Уведомления |

## API

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/api/players` | GET, POST | Список и создание игроков |
| `/api/players/[id]` | GET, PUT, DELETE | CRUD игрока |
| `/api/teams` | GET, POST | Команды |
| `/api/teams/[id]` | GET, PUT, DELETE | CRUD команды |
| `/api/coaches` | GET, POST | Тренеры |
| `/api/coaches/[id]` | GET, PUT, DELETE | CRUD тренера |
| `/api/trainings` | GET, POST | Тренировки |
| `/api/trainings/[id]` | GET, PUT, DELETE | CRUD тренировки |
| `/api/trainings/[id]/attendance` | GET, PUT | Посещаемость |
| `/api/payments` | GET | Список платежей |
| `/api/payments/generate` | POST | Генерация платежей |
| `/api/attendance` | GET | Агрегат посещаемости |
| `/api/schedule` | GET | Расписание |
| `/api/messages` | GET, POST | Сообщения |
| `/api/schools` | GET, POST | Школы |
| `/api/analytics` | GET | Аналитика |
